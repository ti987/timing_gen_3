Quick Fix: Clear Browser Cache

Method 1: Hard Refresh (Fastest)

Press one of these keyboard shortcuts:

  • Windows/Linux: Ctrl + Shift + R or Ctrl + F5
  • Mac: Cmd + Shift + R

Method 2: Using Developer Tools

  1. Press F12 to open Developer Tools
  2. Go to the Network tab
  3. Check the "Disable cache" checkbox
  4. Refresh the page with F5

Method 3: Empty Cache and Hard Reload

  1. Press F12 to open Developer Tools
  2. Right-click the browser's refresh button
  3. Select "Empty Cache and Hard Reload"

What Was Fixed

Problem: Paper.js hit testing only searched the active layer (measureLayer), but AC table elements are in the signalLayer.

Solution: Created hitTestAllLayers() method that searches all 4 layers:

  • backgroundLayer
  • gridLayer
  • signalLayer (where AC tables live)
  • measureLayer

Result: Context menus now work on all AC table cells!
